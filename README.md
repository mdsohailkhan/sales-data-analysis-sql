# Sales Data Analysis using SQL
"Analyze sales transactions using SQL and Python to uncover insights such as top-selling products, best customers, monthly trends, and category performance, providing actionable data-driven recommendations for business decisions."

## Technologies
- SQL (SQLite)
- Python (optional, for data loading and analysis)
- Pandas (optional, for data manipulation)

## Key Tasks
- Calculated total sales per product
- Identified top customers by purchase amount
- Analyzed monthly sales trends
- Evaluated sales performance by product category
- Extracted actionable insights for business decision-making
